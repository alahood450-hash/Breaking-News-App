# Breaking News – Mobile News App

Breaking News is a mobile-first news application designed to deliver real-time breaking stories with a clean dark interface. The app provides a fast and distraction-free reading experience while supporting both readers and journalists.

---

## Project Overview

This project presents the design of a modern news application where users can stay informed with the latest stories while journalists can publish articles directly from the platform.

The design focuses on clarity, speed, and accessibility for mobile users.

---

## Features

### Home Feed
- Live breaking news carousel
- Latest news feed
- Article cards with images and timestamps
- Bookmark feature for saving articles

### Article Detail
- Full-width hero image
- Author information
- Clean reading layout
- Article sharing option

### Categories
News can be filtered by categories:

- Technology
- Business
- Sports
- World
- Local

### Saved Articles
- Bookmark articles for later reading
- Empty state guidance
- Ability to remove saved articles

### Post News (Journalist Feature)

Journalists can create and publish articles directly from the application using:

- Category selection
- Headline and subheadline
- Featured image
- Rich text editor
- Tags system
- Publish or save draft options
